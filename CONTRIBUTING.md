# Contributing

This repository is documentation-first. The most valuable improvements are clarity, reproducibility, and careful distinction between what was completed and what remains future work.

## Guidelines

1. Keep the documented workflow aligned with the attached report and notes.
2. Prefer Markdown guides over leaving critical steps hidden inside raw notes.
3. Clearly separate:
   - completed RF-simulation work
   - LMF experimentation
   - PlutoSDR observations
   - future USRP-based deployment
4. Do not present unverified hardware deployment as completed work.
5. Keep links to external OAI repositories explicit when referring to code not stored here.
