# Project Summary

## Title

Implementation and Analysis of 5G Core and RAN using OpenAirInterface

## Core idea

The project studies how a 5G Core Network, a simulated radio access network, and supporting services such as LMF can be deployed and analyzed using OpenAirInterface in a reproducible software environment.

## Main achievements

- deployed OAI CN5G services with Docker
- ran RF simulation with simulated gNB and UE
- used logs and interface checks to validate registration and session establishment
- explored LMF-related workflows in the OAI environment
- documented PlutoSDR integration limitations honestly

## Scope boundary

The report makes an important distinction:

- completed: software-based RF simulation and software-level OAI integration
- attempted but not finalized: PlutoSDR integration
- future path: USRP-backed real-time deployment

## Why this repo exists

The attached files contained valuable setup knowledge, but much of it was trapped inside a report, PowerPoint slides, and informal notes. This repository reorganizes that knowledge into clearer GitHub-style guides.
