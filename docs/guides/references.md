## What these sources are used for

- The report provides the authoritative academic narrative of the project.
- The raw notes provide the practical command-level workflow.
- The external OAI repositories provide the actual implementation codebase used by the documented setup.

## Important distinction

This repository is a curated project wrapper around the academic artifacts and deployment knowledge. It is not a fork of the full OpenAirInterface implementation itself.
