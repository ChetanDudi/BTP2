# Guides

This folder contains the cleaned, reader-friendly project documentation derived from the report and raw setup notes.

## Files

- `project-summary.md`
  High-level project purpose, scope, and outcomes.

- `system-architecture.md`
  Main 5G components and how they interact.

- `oai-rfsim-workflow.md`
  Step-by-step workflow for the Dockerized OAI RF-simulation setup.

- `lmf-experimentation.md`
  Notes on the LMF-related part of the project.

- `plutosdr-notes.md`
  Summary of why PlutoSDR was explored and why it was not the final path.

- `reproducibility.md`
  What future students can realistically reproduce from this repository.

- `references.md`
  Links to project artifacts and external OAI repositories.

- `repo-name-suggestions.md`
  Suggested GitHub repository names for publishing the project.
