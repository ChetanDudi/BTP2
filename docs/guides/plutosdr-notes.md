# PlutoSDR Notes

## Role in the project

PlutoSDR was explored as a possible intermediate hardware platform before a full USRP-based deployment.

## Outcome

According to the final report:

- PlutoSDR experimentation was attempted
- it was useful for understanding integration limitations
- it did not become the final working deployment path

## Why this matters

This is an important engineering result. The project does not overclaim hardware success; instead, it documents why RF simulation was the completed path and why USRP remains the recommended future hardware route.
